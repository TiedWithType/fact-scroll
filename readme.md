# FactScroll

> Interaktywny projekt przewijany sekcjami, prezentujący ciekawostki w atrakcyjnej wizualnie formie.  
> An interactive scroll-based project that showcases trivia in a visually engaging layout.

---

## 🇵🇱 Opis projektu

**FactScroll** to nowoczesna aplikacja webowa prezentująca 11 sekcji z ciekawostkami z różnych dziedzin (m.in. kosmos, ocean, technologia). Każda sekcja:

- zajmuje pełen ekran (100dvh/100vw),
- zawiera indywidualny kolor tła, tekst i ikonę,
- posiada identyfikator (`id`) widoczny w adresie URL

### 🔧 Funkcje

- 📱 **Responsywność**: układ poziomy na desktopie, pionowy na mobile.
- 📍 **Dynamiczne kropki nawigacyjne**:
  - wskazują aktywną sekcję,
  - są generowane dynamicznie na podstawie liczby sekcji,
  - znikają automatycznie po kilku sekundach bez interakcji.
- 💡 **Optymalizacje wydajnościowe**:
  - preload stylów, czcionek, ikon,

---

## 🇬🇧 Project Description

**FactScroll** is a modern scroll-based web app that presents 11 fullscreen sections with interesting trivia across various topics (e.g., space, ocean, tech). Each section:

- occupies the entire screen (100dvh/100vw),
- features a unique background, text color, and icon,
- includes an `id` reflected in the browser URL,

### 🔧 Features

- 📱 **Responsive Design**: horizontal layout on desktop, vertical on mobile.
- 📍 **Dynamic Dot Navigation**:
  - indicates the active section,
  - generated based on the number of sections,
  - auto-hides after a period of inactivity.
- 💡 **Performance Optimizations**:
  - preload styles, fonts, icons
